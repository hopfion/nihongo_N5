# 日語例句手帖 PWA

## 部署（GitHub Pages，一次即可）
1. 在 GitHub 開一個新 repository（Public）。
2. 把這個資料夾裡的全部檔案上傳到 repository 根目錄。
3. Settings → Pages → Branch 選 main、資料夾選 /(root) → Save。
4. 一兩分鐘後會得到網址 https://你的帳號.github.io/repository名/

## 安裝
- iPhone：Safari 開啟網址 → 分享 → 加入主畫面。
- iMac／MacBook（Safari）：開啟網址 → 檔案 → 加入 Dock。
- iMac／MacBook（Chrome／Edge）：網址列右端的安裝圖示 → 安裝。

## 離線
第一次開啟時自動快取，之後斷網也能完整使用（含朗讀，前提是系統已裝日文語音）。

## 更新
之後拿到新版檔案時，覆蓋上傳到同一個 repository 即可；快取版本號在 sw.js 第一行，新版會自動遞增。
